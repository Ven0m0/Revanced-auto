ReVanced Builder

Automated APK patching system for ReVanced and RVX. Download stock APKs, apply patches, and build optimized APKs — all from a single config file.

Open Configuration Generator View on GitHub

Features

Everything you need to build patched APKs automatically

⚙️

Configuration Generator

Visual config.toml builder with an app catalog, interactive patch selector, live preview, and one-click download. No manual editing required.

Open Generator →
📦

Multi-Source Patches

Combine patches from multiple GitHub repositories. Version detection uses union strategy — the highest version supported by at least one source.

⚡

Build Cache

Smart TTL-based caching with SHA-256 integrity verification. Skip re-downloading unchanged files and reduce build times significantly.

📋

Changelog Automation

Generate comprehensive changelogs from git commits with conventional commit support. Includes patches changelog from GitHub releases.

🔍

Dependency Checker

Automated monitoring of CLI, patches, and APK versions. Daily GitHub Actions checks with automatic issue creation when updates are found.

🚀

Parallel Builds

Build multiple apps simultaneously with configurable parallelism. Download fallback chain: APKMirror, Uptodown, Archive.org.

Quick Start

Get up and running in four steps

1

Configure

Use the configuration generator to create your config.toml visually, or edit it by hand.

2

Select Apps

Pick from the app catalog with pre-filled download URLs, patch sources, and recommended settings.

3

Choose Patches

Browse available patches with the visual selector. Search, filter, and toggle patches with a single click.

4

Build

Run ./build.sh config.toml and the system handles downloading, patching, and signing.
