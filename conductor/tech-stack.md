# Technology Stack

## Core Technologies
-   **Language:** Bash (Version 4.0+)
-   **Runtime:** Java (OpenJDK 21+)
-   **Helper Languages:** Python 3 (Required for TOML parsing via `tomllib` or equivalent)

## Build Tools & Binaries
-   **Android Tools:** `aapt2` (Android Asset Packaging Tool), `apksigner`
-   **Utilities:** `jq` (JSON processor), `zip`, `unzip`
-   **Network:** `curl` or `wget`

## Infrastructure & CI/CD
-   **Platform:** GitHub Actions (Workflows defined in `.github/workflows/`)
